# Suggested Commands

## Project Setup
```bash
# After modifying Project.swift or Tuist helpers
tuist generate

# Full setup
tuist install && tuist generate
```

## Open in Xcode
Open `App.xcworkspace` (not .xcodeproj)

## Git
- Main branch: main
- Create new branch before working if on main
- Create PR when work is done
- Version bump branch format: `a.b.c`
- Version bump commit: `bump: release a.b.c`
